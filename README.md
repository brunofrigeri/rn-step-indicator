# rn-step-indicator

A simple react-native library component for a Step Indicator with custom styling.

## Installation

```sh
npm install rn-step-indicator
```

## Usage

```js
import { multiply } from 'rn-step-indicator';

// ...

const result = await multiply(3, 7);
```
